CPE Virtua Tour & Game Activity

คำอธิบาย

Project CPE Virtual Tour & Game Activity เป็นการสร้างสรรค์ประสบการณ์ใหม่ที่นำเทคโนโลยีล้ำสมัยมายกระดับการเรียนรู้และการสำรวจ โดยใช้เทคโนโลยี ในการออกแบบการนำเสนอข้อมูลและสร้างประสบการณ์ที่น่าสนใจและสนุกสนาน กิจกรรมนี้ออกแบบมาเพื่อให้ผู้เข้าร่วมได้สัมผัสกับการสำรวจที่ทันสมัย มีปฏิสัมพันธ์ได้ และช่วยสร้างความเข้าใจเกี่ยวกับตึกสาขาวิศวกรรมคอมพิวเตอร์ มหาวิทยาลัยราชภัฏพิบูลสงคราม

หน้าที่ที่รับผิดชอบ (Game Developer)

ได้รับหน้าที่ในการสร้างมินิเกมในVirtua Tourเพื่อเพิ่มความสนุกของการสำรวจสาขาวิศวกรรมคอมพิวเตอร์ เกมที่สร้างมี2รูปแบบคือแบบแรกจะเป็นแนวPuzzleในรูปแบบของการต่อจิกซอ เกมที่2คือเกมถอดรหัสด้วยเลขBinaryเพิ่มเพิ่มความท้าทายจากเกมรูปแบบแรก ตัวเกมได้มีการเชื่อฐานข้อมูลMySQLเพื่อทำการเกมข้อมูลชื่อผู้เล่นละแสดงออกมาโชว์ในLeaderboard
